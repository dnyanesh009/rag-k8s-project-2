# RAG Kubernetes Project

## Overview
This project implements a Retrieval-Augmented Generation (RAG) model using Kubernetes orchestration.

## Installation
Follow these steps to set up your environment:
1. Clone the repository:
   ```
   git clone https://github.com/dnyanesh009/rag-k8s-project-2.git
   ```
2. Navigate to the project directory:
   ```
   cd rag-k8s-project-2
   ```

## Usage
To use the RAG model, you can run the following command:
```
kubectl apply -f deployment.yaml
```

## Deployment
Keep in mind to configure your Kubernetes cluster properly before deploying. Use the following command to deploy:
```
kubectl apply -f service.yaml
```
